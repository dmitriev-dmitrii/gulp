
чтобы запустить проект нужно открыть терминал в текущей папке 

у вас должен быть установлен node js.

в терминале открытом в папке проекта прописать  :

npm i 

установятся необходимые пакеты
по окончанию установки прописать в терминале :

gulp 


проект должен запустить с локальным сервером.

что сделать версию  в продакшн нужно прописать в терминале :

gulp build

создастся папка public с сжатыми файлами
